About Me

Welcome to my professional portfolio! I'm Fady Emad Milik, a dedicated and meticulous Software Tester (QA Engineer) with a passion for ensuring the delivery of high-quality, reliable, and user-friendly software solutions. My expertise lies in identifying defects, optimizing testing processes, and contributing to robust product development lifecycles.

My Philosophy

I believe that quality is not just a phase, but an integral part of the entire software development process. My approach combines a keen eye for detail with a comprehensive understanding of testing methodologies to uncover issues early and prevent them from reaching production. I am committed to continuous learning and adapting to new technologies and challenges.

My goal is to provide actionable insights that help teams build better products, fostering a culture of quality and excellence.

E-commerce Platform QA & Automation

Led the quality assurance efforts for a large-scale e-commerce web application, including manual and automated testing.

  • Challenge: Ensure seamless shopping experience and robust transaction processing.
  • My Role: Senior QA Engineer (Test plan creation, automation script development, defect management, regression testing).
  • Methodology/Tools: Agile Scrum, Selenium WebDriver (Java), TestNG, Maven, Jira, Postman for API testing.

Mobile App Performance & Security Testing

Conducted performance and basic security testing for a high-traffic mobile application to ensure scalability and data integrity.

  • Challenge: Identify performance bottlenecks and potential security vulnerabilities.
  • My Role: QA Analyst (Load testing, stress testing, security vulnerability scanning, reporting).
  • Methodology/Tools: JMeter, OWASP ZAP (basic scans), Android Studio Profiler, Fiddler.
  • Process: Scripting load scenarios, analyzing response times, identifying resource utilization spikes, vulnerability assessment.
  • Impact: Identified critical API response delays, leading to a 20% improvement in app responsiveness under load; detected and reported 3 medium-severity security flaws.
